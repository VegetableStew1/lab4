# Онлайн-курс веб-розробки

## Опис
Це односторінковий сайт для онлайн-курсу веб-розробки. Проєкт створений з використанням HTML, CSS та JavaScript.

## Мета роботи
Закріпити навички створення веб-сторінок та додати інтерактивність за допомогою JavaScript.

## Функціонал
- Відображення інформації про курс
- Плавна прокрутка до розділу "Програма"
- Відкриття модального вікна при натисканні кнопок
- Імітація відправки форми (alert)
- Посилання на соціальні мережі

## Структура
- index.html 
- style.css 
- script.js 

## Як запустити
Відкрити файл index.html у браузері

## Технології
- HTML
- CSS
- JavaScript
